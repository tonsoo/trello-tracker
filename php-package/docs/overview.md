# Visão Geral

O Trello Tracker recebe mensagens do WhatsApp e as transforma em ações no Trello usando IA. É distribuído como um pacote Laravel e integra via Service Provider com bindings, rotas e um arquivo de configuração publicável.

- Namespace do pacote: `Tonso\TrelloTracker\`
- Provider de entrada: `src/TrelloTrackerServiceProvider.php`
- Rotas: `routes/api.php`
- Config: `config/trello-tracker.php`
